Wildfires in the U.S.A.

Description

This website hosts an up to date collection of interactive maps that show all the current active wildfires in the US as of 12/9/2020, based on data from the National Wildfire Coordinating Group.

MARKER MAP
This map places markers at the locations of all currently active wildfires, with additional details.

Go to Marker map

HEATMAP
This map illustrates the areas of higher fire concentration.

Go to Heatmap

CHOROPLETH MAP
This map illustrates how each state compares in total acreage currently burning.

Go to Choropleth Map
